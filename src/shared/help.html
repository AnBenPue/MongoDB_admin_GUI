Help document for MongoDB manager

This document will help you to discover all the functionalities of the MongoDB manager. It is possible to view this document selecting Help-> Show help.

Credentials

To load the .json file with the credentials used to access the MongoDB database, select Options-> Credentials.

Change roles

To change the roles of a specific user for a database, select the related role in the table. A drop menu will appear and it is possible to select the desired role. You can repeat the operation for all the users/databases displayed in the table. Once the selection of the role is done, to save the changes it is required to click the Save button or select File-> Save. To discard the changes and reload the roles table click the Reload button or select File-> Reload.

Save changes

To save the changes made to the table and apply them to the database, click the Save button or select File-> Save.

Reload roles

To reload the roles in the table, click the Reload button or select File-> Reload.

Add user

To add a new user, select Settings-> User-> Add. It is possible to add a user by clicking the Add user buton.

Delete user

To delete a user, select Settings-> User-> Delete. It is possible to delete a user by clicking the Delete user buton.

Add database

To add a new database, select Settings-> Database-> Add. It is possible to add a database by clicking the Add database buton.

Delete database

To delete a database, select Settings-> Database-> Delete. It is possible to delete a database by clicking the Delete database buton.

Show logger

A list of the actions performed in the database can be visualized in the logger. To show the logger window, select Help-> Show logger. It is possible to show or hide the INFO messages by clicking the Show INFO flag.
